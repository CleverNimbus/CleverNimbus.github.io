This repository publishes static web builds to the `gh-pages` branch.

Published content on `gh-pages`:

- Root site (landing page): [https://clevernimbus.github.io/](https://clevernimbus.github.io/)
- NoiseLabyrinth app: [https://clevernimbus.github.io/app_noiselabyrinth/](https://clevernimbus.github.io/app_noiselabyrinth/)
- GTFYW internal debugger: [https://clevernimbus.github.io/app_gtfyw_internal/](https://clevernimbus.github.io/app_gtfyw_internal/)

Deployment is performed with the script at `scripts/publish_landingpage.sh`, which builds and publishes the selected app to `gh-pages`.
